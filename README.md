Реализация циклической очереди на C++ с использованием динамической памяти, также реализация используя стандартной библиотекой шаблонов STL и реализация на Python.
Реализация GUI для взаимодействие с циклическая очередь на Python, с использованием tkinter.

Комманды в vs code для создания dll (для MinGW gcc/cpp):
g++.exe -shared -static -static-libgcc -static-libstdc++ -o Circular_Queue1.dll Circular_Queue1.cpp
g++.exe -shared -static -static-libgcc -static-libstdc++ -o Circular_Queue2.dll Circular_Queue2.cpp
